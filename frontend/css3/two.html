# CSS3 Flex布局

# 基本

  1. 设置了flex布局则为flex container，直接子元素为flex item
  2. display 属性为flex或inline-flex可以成为flex container
  3. 注意，设为 Flex 布局以后，子元素的float、clear和vertical-align属性将失效。

TIP

flex-item 默认都是沿着main axis主轴从mani start 开始往mani end 方向排布

# flex-direction

决定了主轴的方向

  1. 默认值为row（从左到右）
  2. row-reverse
  3. column: cross axis 交叉轴 start -> end
  4. column-reverse:

# justify-content

决定了flex items 在main axis上的对齐方式(主轴方向)

  1. 默认flex-start
  2. flex-end
  3. center
  4. space-between:两端对齐，项目之间的间隔都相等。
  5. space-around:flex item两侧的间隔相等。所以，flex item之间的间隔比flex item与边框的间隔大一倍。

# align-items

决定了flex items 在cross axis上的对齐方式(交叉轴方向)

  1. stretch(默认值):如果项目未设置高度或设为auto，将占满整个flex-container的高度。
  2. flex-start:交叉轴的起点对齐。
  3. flex-end:交叉轴的终点对齐。
  4. center:交叉轴的中点对齐。
  5. baseline:项目的第一行文字的基线对齐。

# flex-wrap

1.nowrap（默认）：不换行。 2. wrap：换行，第一行在上方。 3. wrap-reverse：换行，第一行在下方。

# flex-flow

flex-direction || flex-wrap -> (row||nowrap)

# align-content

属性定义了多行flex-item在cross axis的对齐方式。如果flex-item只有一根轴线，该属性不起作用。

# align-self

align-self属性允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

# flex-grow

flex-grow属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。

# order

order属性定义项目的排列顺序。数值越小，排列越靠前，默认为0。

# flex-shrink

flex-shrink属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。

# flex-basis

flex-basis属性定义了在分配多余空间之前，项目占据的主轴空间（main size）。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小。

# flex

flex属性是flex-grow||flex-shrink||flex-basis的简写，默认值为0 1 auto。后两个属性可选。

  • none: 0 0 auto
  • flex: 1 其他为item的flex为默认的0 1 auto 则flex：1 默认占据剩下空间
