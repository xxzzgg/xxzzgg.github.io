# HTML5

# 标签语义化

  • 语义类标签对开发者更为友好，使用语义类标签增强了可读性，即便是在没有CSS的时候，开发者也能够清晰地看出网页的结构，也更为便于团队的开发和维护。
  • 它的文字表现力丰富，更适合搜索引擎检索（SEO），也可以让搜索引擎爬虫更好地获取到更多有效信息，有效提升网页的搜索量，并且语义类还可以支持读屏软件，根据文章可以自动生成目录等等。
  • article:文章主体部分，因为主体部分具有明确的独立性，所以可以用article来包裹。
  • 语义化
    • 不仅仅是标签表面的语义化
    • 属性也要语义化
    <button type="submit">提交</button>
<button type="reset">重置</button>
<button type="menu">菜单</button>
  • 标签
    • 内联:span、strong、em、del、ins、label、a、sub、sup
    • 块级:div、hx、p、address、ul、ol、lo、dl、dt、dd、table、form
    • 内块:input、img、select、textarea、iframe(可以设置宽高但是有内联的特性)

# 表单

form 块级元素 属性 -action提交的地址 -method get、post -name 数据的名称

  • value数据的值 阻止表单的默认提交
<form action="" method="post" onsubmit="return false">
先去执行 return 后面的内容,如果返回false则会阻止默认表单提交操作
onsubmit默认会提交表单

<button type="submit" onclick="submitForm()">提交</button>

function submitForm (e) {
  var e = e || window.event;
  e.preventDefault();
}
