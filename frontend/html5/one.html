# Canvas

<canvas id="canvas1" width="300" height="150"></canvas>

# 渲染上下文

    const canvas = document.getElementById('canvas1');
    const ctx = canvas.getContext('2d');
    ctx.fillStyle = 'orange';
    ctx.fillRect(25, 25, 100, 100);
    //fillRect(x, y, width, height)
    //绘制一个填充的矩形 坐标为(25, 25) height:100 width:100
    ctx.clearRect(45, 45, 60, 60);
    //clearRect(x, y, width, height)
    //清除指定矩形区域，让清除部分完全透明。
    ctx.strokeRect(60, 60, 30, 30);
    //strokeRect(x, y, width, height)
    //绘制一个矩形的边框
    ------------------------
    ctx.restore();
    //还原上次 保存(save)的状态
    ctx.scale();
    //缩放
    ctx.save();
    //保存状态
    ctx.rotate();
    //canvas 原点来旋转

# 绘制路径

步骤:

  1. 首先，你需要创建路径起始点。
  2. 然后你使用画图命令去画出路径。
  3. 之后你把路径封闭。
  4. 一旦路径生成，你就能通过描边或填充路径区域来渲染图形。
  let canvas = document.getElementById('canvas1');
    let ctx = canvas.getContext('2d');
    ctx.beginPath();
    //首先，你需要创建路径起始点。
    ctx.strokeStyle = 'orange';
    ctx.lineWidth = 10;
    ctx.moveTo(50, 50);
    //将笔触移动到指定的坐标x以及y上。
    ctx.lineTo(100, 50);
    //lineTo(x, y) 绘制一条从当前位置到指定x以及y位置的直线。
    ctx.lineTo(75, 100)
    ctx.lineTo(50, 50);
    ctx.closePath();
    //路径封闭。
    ctx.stroke();
    //一旦路径生成，你就能通过描边或填充路径区域来渲染图形。

# 圆弧

绘制圆弧或者圆，我们使用arc()方法

arc(x, y, radius, startAngle, endAngle, anticlockwise);
// 画一个以（x,y）为圆心的以radius为半径的圆弧（圆），从startAngle开始到endAngle结束，按照anticlockwise给定的方向（默认为顺时针）来生成。逆时针true、顺时针false默认
arcTo(x1, y1, x2, y2, radius)
// 根据给定的控制点和半径画一段圆弧，再以直线连接两个控制点。

TIP

arc()函数中表示角的单位是弧度，不是角度。角度与弧度的js表达式: 弧度= (Math.PI/180)*角度 Math.PI= π = 3.14159 = 180° 按照anticlockwise给定的方向（默认为顺时针）来生成。 逆时针true、顺时针false默认

# 贝塞尔曲线

二次贝塞尔曲线

quadraticCurveTo(cp1x, cp1y, x, y)
// 绘制二次贝塞尔曲线，cp1x,cp1y为一个控制点，x,y为结束点。
ctx.quadraticCurveTo(50, 0, 100, 0)//

三次贝塞尔曲线

bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y);
// 绘制三次贝塞尔曲线，cp1x,cp1y为控制点一，cp2x,cp2y为控制点二，x,y为结束点。
ctx.bezierCurveTo(50, 0, 100, 0, 150, 100)

# 矩形

rect(x, y, width, height)
// 绘制一个左上角坐标为（x,y），宽高为width以及height的矩形。

TIP

当该方法执行的时候，moveTo()方法自动设置坐标参数（0,0）。 也就是说，当前笔触自动重置回默认坐标。
