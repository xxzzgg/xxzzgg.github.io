# React和Babel

默认react可以不使用Babel Babel会把 JSX转译成一个名为React.createElement()函数调用。

# 声明式编程

它允许我们只需要维护自己的状态，当状态改变是，React可以根据最新的状态去渲染页面

ui = fn(state) //fn -> render

var elem = <h2>Hello World</h2>
ReactDOM.render(elem, document.getElementById('app'))
</script>
//必须引用babel和script type="text/babel" 才能使用JSX

# JSX注释：

{/* 我是一段注释 */}

# JSX嵌入变量、数据 {}

在{}中可以正常显示的内容 string number array 
运算符表达式 三目运算符 函数调用都可以
name: "zhangsan"//string 
age: 18 //number
names: ["abc", "ddd", "eee"] //array

undefined null 和布尔类型 均不显示

对象也不能作为jsx的子元素 会直接报错

TIP

  • JSX只能有一个根元素（Vue模板）
  • 通常在JSX外面包裹一个()
  • 单标签需要闭合
  • 布尔类型或 null。什么都不渲染。（主要用于支持返回 test && <Child/>的模式，其中 test 为布尔类型。)

# 事件绑定

  1. 手动bind，改变this执行
  2. 定义函数时使用箭头函数
  3. 在函数里面使用箭头函数(推荐)

# JSX简单实用

 class App extends React.Component{
      constructor () {
        super();
        this.state = {
          count: 0,
          names: ["abc", "ddd", "eee"],
          age: 18   
        }
      }

      render() {
        return (
          <div>
          <h2>当前计数{this.state.count}</h2>
          <ul>
            <li>{this.state.names[0]}</li>
            <li>{this.state.names[1]}</li>
            <li>{this.state.names[2]}</li>
          </ul>
          </div>
        )
      }
    }

  ReactDOM.render(<App/>,document.getElementBy('app'))

    class App extends React.Component{
      constructor () {
        super();
        this.state = {
          count: 0,
          names: ["abc", "ddd", "eee"],
          age: 18
        }
      }

      render() {
        return (
          <div>
          <h2>当前计数{this.state.count}</h2>
          <ul>
            {this.state.names.map((item) => {
              return <li>{item}</li>
            })}
            {/* forEach 不可以 因为forEach 没有返回值 */}
          </ul>
          </div>
        )
      }
    }
    ReactDOM.render(<App/>,document.getElementByI('app'))

TIP

forEach方法与map方法很相似，也是对数组的所有成员依次执行参数函数。 但是，forEach方法不返回值，只用来操作数据。这就是说， 如果数组遍历的目的是为了得到返回值，那么使用map方法，否则使用forEach方法。

# JSX原理

jsx仅仅只是 React.createElement(component, props, ...children)函数的语法糖。

<div style={{display:"block"}}
  onClick = {e => {this.btnClick()}}
  >
	DIVdivdivdiv
  {this.state.isLogin ? "退出" : "登录"}
</div>
//babel
/*#__PURE__*/
React.createElement("div", {
  style: {
    display: "block"
  },
  onClick: e => {
    this.btnClick();
  }
}, "DIVdivdivdiv", this.state.isLogin ? "退出" : "登录");
